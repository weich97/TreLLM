## Reddit / Community Post

```text
I open-sourced TradeArena, a benchmark/audit framework for LLM trading agents.

The goal is not to claim that an LLM can beat the market. The goal is to make
agent decisions reproducible and inspectable under realistic execution and risk
constraints: fees, slippage, latency, liquidity, partial fills, rejected orders,
pre-trade risk gates, post-trade attribution, and replayable trajectories.

First run is API-free:
python -m pip install -e ".[dev]"
python scripts/run_showcase.py

I would especially like feedback on the audit schema, execution realism presets,
and community benchmark manifest design.
```
